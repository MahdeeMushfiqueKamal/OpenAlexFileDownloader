# OpenAlex Downloader Using Selenium

### Installation: 
Install Poetry: 

You can install Poetry using the following command on Linux, macOS, Windows (WSL):

```bash
curl -sSL https://install.python-poetry.org | python3 -
poetry --version
```

Install the Necessary Packages: 
```bash
poetry install 
```

### Running the OpenAlex Downloader
```bash
poetry run python main.py
```
